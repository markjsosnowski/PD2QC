PD2QC: Payday 2 Quickchat Mod Changelog

Version 1.2

2/14/2021

  • Thank you for over 100 downloads! Last update for a while, the mod is pretty much complete.
  • Added Chinese language support. Thanks Arknights!
  • Unofficial Simplified Chinese localization mod is required to use this option.
  • Fixed logic relating to changing multiple menu options on the fly
  • Persist option now correctly shows the menu right when you start a mission when turned on.
  • HUD Panel now better aligns to the side of the screen (caused text to be cut off in some localizations).

Version 1.1

2/9/2021

  • Mod options menu now avalible
  • Added an option to disable quickchat in pause menu (enabled by default)
  • Added an option to adjust the timout for selecting a chat
  • HUD Position and Persistant HUD options are now availible in the mod options menu
  • Added localization support
  • Added 3 language options accessible from the mod options menu.
  • Language options added are English, poorly translated Spanish, and ???
  • Adjusted the size calculation for the HUD popup
  • Custom keybinds are now show in HUD instead of generic directions

  • Dynamically created HUD popup
